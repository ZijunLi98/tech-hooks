## ✨ 特性

- 可靠的代码健壮：使用 Typescript 构建，提供完善的类型定义文件
- 完善的文档能力：支持文档记录，支持 demo 演示
- 完整的测试用例：配套完整的测试用例，帮助您提升项目健壮性

## 📦 安装

```bash
$ npm install --save tech-hooks
# or
$ pnpm add tech-hooks
# or
$ yarn add tech-hooks
```

## 📦 CDN 引入

```html
<script src="https://www.unpkg.com/tech-hooks"></script>
```

## 🔨 使用

```ts
import { useToggle } from 'techHooks';
```
